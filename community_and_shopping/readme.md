## 1. 如何运行


### 1.1 开发环境配置

安装node

### 1.2 运行

#### 1.2.1 前端
```sh
# 打开前端工程目录
cd .\mycommunity\

# 安装依赖
npm install

# 运行开发版
npm run serve

# 构建发布版
npm run build
```
#### 1.2.2
>Server 用于服务后端某些不支持的接口
```sh
cd .\Server\

npm install 

npm start
```
